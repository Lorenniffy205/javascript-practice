# Day 1 — JavaScript practice

## Goal

Learn JavaScript variables, data types, and simple code flow.

## Tasks for today

- Run `npm run day1`
- Read the output in the terminal
- Edit `src/day1/day1.js` and change values or add a new variable
- Save and run again

## Practice items

- `const` for values that do not change
- `let` for values that change
- objects and arrays
- destructuring
- functions that return strings

## Why this helps

This is the real-time way to learn: write code, run code, see the result, change code.

## Next step

After Day 1, move to Day 2 and start learning:

- functions in depth
- arrays methods: `map`, `filter`, `reduce`
- browser console and simple DOM experiments
